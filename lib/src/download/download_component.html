SheetyGUI

SheetyGUI provides the graphic interface to manage your documents and have near full functionality locally.

This is made using Flutter Desktop Embedding, meaning it is not fully production-ready. It supports Windows, Mac, and Linux, with precompiled binaries available for Windows, other platforms coming soon. Mac and Linux will require running from sources. In the future, this will be a single binary for respective platforms.
